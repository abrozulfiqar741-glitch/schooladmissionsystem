# School Admission System

A professional Android application built with **Jetpack Compose** and **Room Database** to manage school student applications.

## 🚀 Features

- **Student Application Form**: Easily enter student names, ages, and grades.
- **Local Persistence**: Uses **Room Database** to save applications permanently on the device.
- **Real-time Search**: Quickly find students in the list with a dynamic search bar.
- **Academic Theme**: A professional **Navy Blue** design system for a clean and focused user experience.
- **Responsive List**: View all submitted applications in a scrollable list with the ability to delete entries.

## 🛠️ Tech Stack

- **UI**: Jetpack Compose (Material 3)
- **Database**: Room (with KSP)
- **Architecture**: MVVM (Model-View-ViewModel) with Repository pattern
- **Asynchronous Code**: Kotlin Coroutines & Flow
- **Dependency Management**: Version Catalogs (`libs.versions.toml`)
